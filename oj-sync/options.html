OJ Sync

Companion to CP Helper in Visual Studio Code (official Microsoft build). The toolbar button POSTs extracted JSON to CP Helper on your machine. Nothing is copied to the clipboard. Keep localhost HTTP enabled so Chrome does not need a vscode:// prompt for normal imports.

Local HTTP (recommended)

Must match CP Helper settings cp-helper.localImportPort (default 17337) and path /import. CP Helper must be running in VS Code with the local import server enabled. If the POST fails, you get a red badge; enable the fallback below to open the editor.

VS Code default above. This does not import samples — use it to jump to CP Helper when localhost is down. Manual JSON paste remains in CP Helper’s Samples view.
